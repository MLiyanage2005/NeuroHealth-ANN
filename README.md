# NeuroHealth-ANN
NeuroScope-AI is an AI-assisted cognitive wellness assessment platform designed to analyze cognitive and behavioral indicators such as memory issues, sleep quality, focus problems, stress levels, and confusion frequency.

The system uses an Artificial Neural Network (ANN) built with TensorFlow to predict cognitive wellness risk scores and classify users into four cognitive risk levels.

Based on the predicted level, the platform provides supportive wellness suggestions, cognitive health guidance, and alerts encouraging users to seek professional medical consultation when higher-risk indicators are detected.

This project is intended for educational and research purposes and does not provide medical diagnoses.
